# ricoleehduu.github.io

Personal academic homepage of Zhi Li (李智).

## Deploy

1. Create a GitHub repository named `ricoleehduu.github.io`
2. Push the `index.html` to the `main` branch
3. Enable GitHub Pages (Settings → Pages → Source: main branch)
4. Visit `https://ricoleehduu.github.io`

## Update

Replace the `index.html` with your latest information and push.
